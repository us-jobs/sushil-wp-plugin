Automatically writes SEO-optimized articles with featured images from Freepik.

== Description ==

RankReady AI is a powerful WordPress plugin designed to automate your content creation process. It leverages AI to generate high-quality, SEO-optimized articles and automatically pairs them with relevant featured images from Freepik.

Keywords: SEO, AI Article Writer, Automated Content, Freepik Images, WordPress Automation.

== Installation ==

1. Upload the `rankready-ai` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your settings in the RankReady AI menu.

== Frequently Asked Questions ==

= Does it require an API key? =

Yes, you will need to configure your respective (completely free) API keys for AI and Freepik in the settings.
